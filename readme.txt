	- Compiler le fichier .c dans l'invite de commande avec gcc main.c -o main
	- Lancer le fichier compiler avec ./main
	- Pour jouer :
		- 4 vers la gauche
		- 6 vers la droite
		- 8 vers le haut
		- 2 vers le bas
		- Objets :
			-1==FLEUR
			-2==ARBRE
			-3==ROCHER
			-4==CLEF
			-5==PIECE D'OR
			-6==CADENAS
			-7==PIEGE
			-8==MONSTRE
		- Pour finir :
			- SI 0 PV ALORS PARTIE PERDU
			- SI 10 PO ALORS PARTIE GAGNE
			- SI ON TAPE 0 ALORS FIN DE LA PARTIE
